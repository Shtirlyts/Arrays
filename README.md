# Работа с массивами в C

## 📋 Задания к лаб. 11
1. **Заполнение массива с консоли** - ввод элементов массива с клавиатуры
2. **Преобразование массива** - инвертирование знаков всех элементов
3. **Печать массива** - вывод элементов массива на экран
4. **Подсчет элементов** - подсчет элементов массива, значения которых больше заданного числа A

## 📋 Задания к лаб. 14
1. **Заполнение массива значениями** - заполнение массива квадратами индексов (f(i) = (i+1)²)
2. **Печать массива** - вывод элементов массива на экран
3. **Вычисление среднего значения** - вычисление среднего значения элементов, расположенных между минимальным и максимальным значением следующих после него элементов

## 📋 Задания к лаб. 16
1. **Генерация трех массивов** - создание массивов a, b, c со случайными вещественными числами (10-50 элементов)
2. **Создание массива d** - формирование массива d по правилам:
   - Первый элемент: среднее значение массива b
   - Последний элемент: среднее значение массива c
   - Средние элементы: элементы массива a (начиная с первого положительного), преобразованные по правилу: если >1 → -0.5, иначе → +0.5

## 🔧 Реализованные функции

### Лабораторная работа 11

#### 1. `double* input_array(double* ptr_array, int n)`
**Назначение**: Ввод элементов массива с клавиатуры  
**Параметры**:
- `ptr_array` - указатель на массив
- `n` - размер массива  
**Возвращает**: указатель на заполненный массив

#### 2. `int print_array(double* ptr_array, int n)`
**Назначение**: Вывод элементов массива на экран  
**Параметры**:
- `ptr_array` - указатель на массив
- `n` - размер массива  
**Возвращает**: количество выведенных элементов

#### 3. `int count_greater_than_A(double* ptr_array, int n, double A)`
**Назначение**: Подсчет элементов массива, значения которых больше заданного числа A  
**Параметры**:
- `ptr_array` - указатель на массив
- `n` - размер массива
- `A` - число для сравнения  
**Возвращает**: количество элементов, больших A

### Лабораторная работа 14

#### 4. `double* full_elements(double* ptr_array, int n)`
**Назначение**: Заполнение массива квадратами индексов (f(i) = (i+1)²)  
**Параметры**:
- `ptr_array` - указатель на массив
- `n` - размер массива  
**Возвращает**: указатель на заполненный массив

#### 5. `double average_between_min_max(double* arr, int n)`
**Назначение**: Вычисление среднего значения элементов, расположенных между минимальным и максимальным значением следующих после него элементов  
**Параметры**:
- `arr` - указатель на массив
- `n` - размер массива  
**Возвращает**: среднее арифметическое значение элементов между min и max

### Лабораторная работа 16

#### 6. `double* create_array_d()`
**Назначение**: Создание массива `d` по правилам лабораторной работы 16  
**Алгоритм**:
1. Генерирует три массива `a`, `b`, `c` случайных размеров (10-50 элементов) с случайными вещественными числами
2. Находит первый положительный элемент в массиве `a`
3. Преобразует элементы `a` (начиная с первого положительного): если >1 → -0.5, иначе → +0.5
4. Создает массив `d`: первый элемент = среднее `b`, последний = среднее `c`, средние = преобразованные элементы `a`  
**Возвращает**: указатель на созданный массив `d`
